# nokia.nsp.nsp httpapi plugin

**HttpApi Plugin for Nokia NSP**

## Synopsis

* This HttpApi plugin provides OAuth2 client credentials authentication for Nokia NSP.
* Automatically handles token acquisition and revocation.
* Supports standard Ansible HTTPAPI module interface.

## Connection Options

| Parameter | Type | Comments |
| --- | --- | --- |
| nsp_host {#param-nsp-host} | str | NSP server hostname or IP address.<br/>name: ANSIBLE_HTTPAPI_NSP_HOST<br/>name: ansible_httpapi_nsp_host |

*New in version "0.0.1"*
